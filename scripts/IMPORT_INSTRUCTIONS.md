# 🚀 Product Import Instructions

## 📦 How to Import Products to Your FireShop

### **Step 1: Go to Inventory Page**
1. Login as admin
2. Navigate to `/inventory` 
3. You'll see the "Import JSON" button

### **Step 2: Import Your Products**
1. Click the **"Import JSON"** button
2. Select the `products.json` file from `/scripts/` folder
3. Wait for the import to complete

### **Step 3: Save to Firebase**
1. After import, you'll see all products marked as "New"
2. Click the **"Save Changes"** button
3. Wait for all products to be saved to Firebase

## 🆘 Troubleshooting

### **Import Fails:**
- Check JSON file format
- Ensure file is not corrupted
- Try importing in smaller batches

### **Products Not Showing:**
- Click "Save Changes" after import
- Refresh the page
- Check browser console for errors

### **Wrong Categories:**
- Edit products individually
- Update category field
- Save changes
